# Are you okay

前端使用uni-app框架，后端使用Beego框架

1. **极简注册与“守护人”设置**
    - 微信一键登录，降低使用门槛。
    - **MVP核心功能**：引导用户设置1位“心灵守护人”（即紧急联系人），填写姓名、电话。**关键点**：需明确告知用户，在什么情况下（如连续情绪极低）会通知守护人，并获得用户明确授权。这比“活着么”的机制更需谨慎和温度。
2. **每日情绪签到（核心互动）**
    - **主界面**：一个非常简单的按钮或卡片，如“今天，你还好么？”。
    - **情绪记录**：
        - **方式一（极简）**：选择1-3个代表当下情绪的Emoji（😊 😐 😢 😤 🥱），并可选填一句简单的想法或日记（限制字数，如140字）。
    - **完成奖励**：获得一句随机的、温暖的肯定语或小诗（如：“谢谢你照顾自己的感受，今天也辛苦了。”）。建立即时正反馈。
3. **基础情绪关怀与预警**
    - **连续签到鼓励**：连续签到3天、7天有特别鼓励语。
    - **消极情绪预警**：**MVP核心逻辑**——当系统检测到用户**连续2-3天选择高度负面情绪（如悲伤、愤怒的低分值）** 时，触发关怀流程：
        1. 首先向**用户本人**推送一条特别关怀信息：“注意到你最近几天情绪有些低落，我们为你准备了一段放松音频，希望能让你感觉好一点。” 并附上一个简单的疗愈资源链接。
        2. 如果用户**连续超过3天（可设置）** 持续负面情绪且未与小程序互动，系统可**在次日**向用户的“心灵守护人”发送一条**温和、非 alarming 的短信通知**。**文案至关重要**，例如：“您好，[用户昵称]在‘还好么’小程序中已连续多日记录到较低落的情绪。Ta可能最近需要多一些关心和陪伴。建议您可以找个合适的时间，轻松地和Ta聊聊天。**——‘还好么’温情提醒”**。
4. **轻量疗愈内容库（核心价值）**
    - **形式**：5-10分钟的音频为主（制作成本低，体验沉浸）。
    - **内容**：提供4-5个类别的音频，每个类别3-5条。
        - 正念呼吸引导
        - 身体放松扫描
        - 缓解焦虑的短故事
        - 积极心理暗示语录
        - 白噪音（雨声、篝火声）
    - **呈现**：在用户记录负面情绪后主动推荐，也有一个简单的“疗愈资源”页面可以自由选择。
